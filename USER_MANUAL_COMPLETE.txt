📊 完整操作流程示例

【示例：钢管检测完整流程】

第1步：打开系统
   → 浏览器访问网址
   → 检查顶部状态栏绿灯亮起
   → 确认"系统运行"和"数据库"都正常

第2步：配置参数
   → 点击右侧"设置"按钮
   → 填写工件编号：例如 "GP-20251009-001"
   → 选择材质：Q235
   → 输入规格：Φ219×6mm
   → 设置采样频率：2000 Hz
   → 调整灵敏度：7
   → 设置上门限：80%
   → 设置下门限：20%
   → 保存配置

第3步：启动检测
   → 返回主界面
   → 点击左侧橙色"▶"按钮
   → 按钮变为"⏸"
   → 状态显示"监测中"
   → 波形区域开始显示绿色实时波形

第4步：观察波形
   → 查看黑色背景的波形显示区
   → 绿色波形实时更新
   → 底部黄色标尺显示位置（0-100mm）
   → 如有缺陷，波形会超出红色区域
   → 顶部"缺陷: N"计数器会更新

第5步：停止监测
   → 检测完成后，点击"⏸"按钮
   → 状态变为"已停止"
   → 波形停止更新

第6步：保存数据
   → 点击右侧"SAVE"按钮
   → 系统保存到数据库
   → 显示"数据保存成功"提示

第7步：查看结果
   → 点击左侧"历史"按钮
   → 在列表中找到刚才的记录
   → 点击"查看"按钮查看详情

第8步：生成报告
   → 在历史记录中点击"生成报告"
   → 选择"标准报告"
   → 填写报告标题
   → 点击"生成"
   → 自动下载PDF报告
