📋 系统概述

关于本系统

磁检测仪器网页版是一套专业的无损检测系统，用于检测金属材料（如钢管、钢板）中的表面和近表面缺陷。系统采用现代化的Web技术，提供直观的操作界面和强大的数据分析功能。

🚀 快速开始

首次使用流程

步骤 1: 打开系统

在浏览器中访问系统网址，系统会自动加载并初始化数据库连接。

✅ 检查连接状态
顶部状态栏会显示"系统运行"和"数据库"两个绿色指示灯，表示系统正常运行。
步骤 2: 配置检测参数

首次使用建议先设置检测参数：

  1. 点击右侧的"设置"按钮
  2. 配置工件信息（编号、材质、规格）
  3. 设置检测参数（采样频率、灵敏度）
  4. 调整门限值（上门限 80%，下门限 20%）
  5. 保存配置
步骤 3: 启动检测

返回主界面，进行首次检测：

  1. 确认所有参数设置正确
  2. 点击左侧的"▶"播放按钮
  3. 系统开始实时监测，波形区域会显示动态数据
  4. 如检测到缺陷，会自动标记并计数
  5. 点击"⏸"暂停按钮停止检测
步骤 4: 保存数据

检测完成后保存数据：

  1. 停止检测后，点击右侧"SAVE"按钮
  2. 系统会将测试记录保存到数据库
  3. 保存成功后会显示提示消息
步骤 5: 查看结果

查看和分析测试结果：

  1. 点击"历史"按钮查看历史记录
  2. 点击"报告"按钮生成检测报告
  3. 使用左侧菜单"📈 数据分析"查看统计图表
⚠️ 注意事项
  • 首次使用前请确保已完成设备校准
  • 检测过程中避免频繁调整参数
  • 重要数据请及时保存
  • 定期查看系统日志，确保数据库连接正常

📡 实时监测操作指南

准备工作

  1. 确认设备连接正常
  2. 检查校准是否在有效期内
  3. 准备好待检测工件
  4. 设置好检测参数

开始检测

1. 设置基本信息

在主界面底部控制栏设置：

  • 灵敏度：使用滑块调节（范围 1-10），推荐值 7
  • 上门限：输入数值（推荐 80%）
  • 下门限：输入数值（推荐 20%）
💡 参数建议
工件类型 灵敏度 上门限 下门限
钢管（标准） 7 80% 20%
钢板焊缝 8 75% 25%
精密检测 10 70% 30%
2. 启动监测
  1. 点击左侧的橙色"▶"按钮
  2. 按钮变为"⏸"，状态显示为"监测中"
  3. 波形显示区开始显示实时数据
  4. 系统自动运行缺陷检测算法
3. 波形观察

波形显示区说明：

  • 黑色背景：主显示区域
  • 绿色波形：实时磁信号波形
  • 黄色标尺：显示位置刻度（0-100mm）
  • 红色区域：超过上门限的信号（可能缺陷）
  • 网格线：50mm × 50mm 参考网格
4. 缺陷识别

系统会自动识别缺陷并显示：

  • 顶部"缺陷: N"计数器会实时更新
  • 检测到缺陷时会显示警告消息
  • 缺陷信息包括：类型、位置、严重程度
示例输出： 检测到表面裂纹 - moderate 位置: 45.2mm - 46.8mm 峰值幅度: 85.3%
5. 停止监测
  1. 点击"⏸"暂停按钮
  2. 波形停止更新
  3. 状态显示为"已停止"
  4. 此时可以保存数据或重新开始

实时调整参数

在监测过程中可以实时调整灵敏度：

  • 移动底部的灵敏度滑块
  • 或点击右下角的橙色旋钮进行调节
  • 参数变化会立即生效
⚠️ 重要提示
  • 监测过程中频繁调整参数可能影响检测准确性
  • 如发现异常波形，建议停止检测并检查设备
  • 每次检测完成后及时保存数据

📊 历史数据管理

查看历史记录

访问历史数据

有三种方式进入历史数据页面：

  1. 点击左侧"历史"按钮
  2. 点击左侧菜单"📊 历史数据"
  3. 直接访问 /pages/history.html

数据列表

历史数据页面以表格形式显示所有测试记录：

列名 说明
测试日期 记录创建的时间
工件编号 被检测工件的唯一标识
材质 工件材质（如 Q235、20#钢）
测试类型 检测类型（钢管、焊缝等）
缺陷数量 检测到的缺陷总数
测试结果 合格/不合格/待定
操作 查看详情、生成报告、删除

筛选和搜索

使用筛选功能

页面顶部提供多种筛选条件：

  1. 搜索框：输入工件编号进行模糊搜索
  2. 日期范围：选择开始和结束日期
  3. 测试结果：选择合格/不合格/待定/全部
  4. 操作员：按操作员姓名筛选

查看详情

详细信息查看

点击某条记录的"查看"按钮，可以看到：

  • 完整的测试参数
  • 波形数据图表
  • 缺陷列表详情
  • 操作员备注

数据导出

支持多种导出格式：

  • Excel：导出表格数据
  • PDF：生成格式化报告
  • CSV：原始数据导出
💡 数据管理建议
  • 定期清理过期数据，释放存储空间
  • 重要数据及时导出备份
  • 使用有意义的工件编号便于检索

📈 数据分析功能

统计图表

系统提供多种可视化图表：

1. 缺陷分布图

显示不同类型缺陷的数量分布：

  • 表面裂纹
  • 内部裂纹
  • 气孔
  • 夹杂
  • 折叠
  • 重皮

2. 合格率趋势

按时间显示测试合格率变化趋势：

  • 日度统计
  • 周度统计
  • 月度统计

3. 材质分析

不同材质的检测结果对比

使用分析工具

步骤 1: 选择时间范围

在页面顶部选择要分析的时间段：

  • 最近7天
  • 最近30天
  • 最近3个月
  • 自定义时间范围
步骤 2: 选择分析类型

选择需要查看的图表类型：

  • 缺陷统计
  • 合格率分析
  • 材质对比
  • 操作员统计
步骤 3: 查看和导出

分析结果展示后可以：

  • 放大/缩小图表
  • 查看具体数据点
  • 导出图表为图片
  • 打印分析报告

关键指标

指标名称 计算方式 参考值
合格率 合格数 / 总测试数 × 100% ≥ 95%
平均缺陷数 总缺陷数 / 测试次数 ≤ 2 个/次
严重缺陷率 严重缺陷 / 总缺陷 × 100% ≤ 5%

📄 报告生成与管理

报告类型

📋
标准报告
包含完整测试数据和波形图
📄
简化报告
仅包含关键信息和结论
📊

批量数据的统计分析报告

生成报告

单个测试报告
  1. 在历史数据页面找到目标记录
  2. 点击"生成报告"按钮
  3. 选择报告类型（标准/简化）
  4. 填写报告标题和备注
  5. 点击"生成"
  6. 系统生成PDF并自动下载
批量报告
  1. 在报告管理页面点击"批量生成"
  2. 设置时间范围或选择多条记录
  3. 选择报告模板
  4. 点击"批量生成"
  5. 等待生成完成
  6. 打包下载所有报告

报告内容

标准报告包含：

  • 报告编号和生成日期
  • 工件基本信息
  • 检测参数设置
  • 波形图像
  • 缺陷列表详情
  • 检测结论
  • 操作员签名
  • 审核人签名

报告管理

在报告管理页面可以：

  • 查看所有已生成的报告
  • 重新下载历史报告
  • 删除过期报告
  • 预览报告内容
📝 报告规范
  • 报告编号格式：年月日-序号 (如 20251009-001)
  • 所有报告需要操作员签名确认
  • 重要报告需要审核人复核签字
  • 报告保存期限不少于3年

工件信息

填写工件信息

在设置页面填写待检测工件信息：

  • 工件编号：唯一标识码（必填）
  • 材质：如 Q235、20#钢、304不锈钢
  • 规格：如 Φ219×6mm、500×300×10mm
  • 测试类型：选择钢管检测、焊缝检测等
  • 操作员：当前操作人员姓名

测试模板

系统预置了常用的测试模板：

  • 钢管标准检测：采样率2000Hz，灵敏度7
  • 钢板焊缝检测：采样率5000Hz，灵敏度8
  • 高灵敏度检测：采样率10000Hz，灵敏度10
使用模板
  1. 在设置页面点击"加载模板"
  2. 选择合适的模板
  3. 系统自动填充所有参数
  4. 可以根据需要微调参数
  5. 点击"保存"应用设置
创建自定义模板
  1. 按需求配置所有参数
  2. 点击"另存为模板"
  3. 输入模板名称和描述
  4. 点击"保存"
  5. 模板会出现在模板列表中

高级设置

专业用户可以调整高级参数：

  • 噪声抑制级别
  • 滤波器参数
  • 触发阈值
  • 自动保存间隔
⚠️ 参数调整注意
  • 不建议随意修改高级参数
  • 参数变更后建议先进行测试验证
  • 重要配置变更应记录备案

❓ 常见问题解答

基础问题

Q1: 系统无法连接数据库怎么办？

A:

  1. 检查网络连接是否正常
  2. 查看顶部状态栏"数据库"指示灯是否为绿色
  3. 刷新页面重新加载
  4. 如仍有问题，请联系技术支持

Q2: 波形显示异常或无波形？

A:

  1. 确认已点击"▶"启动监测
  2. 检查采样频率是否设置过低
  3. 检查浏览器控制台是否有错误信息
  4. 尝试调整灵敏度参数

Q3: 如何提高检测灵敏度？

A:

  • 增加灵敏度参数值（7→8→9→10）
  • 提高采样频率
  • 调整门限范围
  • 使用"高灵敏度检测"模板

Q4: 数据保存失败怎么办？

A:

  1. 检查数据库连接状态
  2. 确认网络连接正常
  3. 检查浏览器控制台错误信息
  4. 尝试重新保存
  5. 必要时可以导出数据到本地

技术问题

Q5: 支持哪些浏览器？

A: 推荐使用以下浏览器的最新版本：

  • Google Chrome（推荐）
  • Microsoft Edge
  • Mozilla Firefox
  • Safari（MacOS）

Q6: 数据存储在哪里？

A: 所有数据存储在 Supabase 云数据库中，具有以下特点：

  • 自动备份，数据安全
  • 支持多用户同时访问
  • 可从任何设备访问
  • 数据加密传输

Q7: 如何导出历史数据？

A:

  1. 进入历史数据页面
  2. 使用筛选条件选择需要导出的数据
  3. 点击"导出"按钮
  4. 选择导出格式（Excel/CSV/PDF）
  5. 下载到本地

Q8: 校准有效期到期怎么办？

A:

  1. 进入"校准管理"页面
  2. 查看当前校准状态
  3. 使用标准试块进行重新校准
  4. 输入新的校准数据
  5. 设置新的有效期
  6. 保存校准记录

操作问题

Q9: 如何删除错误的测试记录？

A:

  1. 在历史数据页面找到要删除的记录
  2. 点击操作栏的"删除"按钮
  3. 确认删除操作
  4. 记录将被永久删除（无法恢复）

Q10: 如何修改已保存的测试记录？

A: 已保存的测试记录原则上不可修改，以保证数据完整性。如需修改，只能：

  • 删除原记录
  • 重新进行测试
  • 保存新记录
